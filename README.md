# NeuroApprendre V5 - MEGA Edition

## 🎯 Plateforme Éducative Complète

- **1000+ Questions** générées dynamiquement
- **20+ Matières** (Français, Maths, Langues, Sciences, etc.)
- **3 Niveaux** (6ème, 4ème, 3ème)
- **Design Pastel** moderne et responsif

## 🚀 Déploiement

### Sur Netlify (Recommended)
1. Push sur GitHub
2. Connecte le repo à Netlify
3. Deploy automatique

### Installation Locale
```bash
# Ouvre simplement index.html dans ton navigateur
open index.html
```

## 📝 Structure

```
neuroapprendre-v5/
├── index.html      # Application complète
├── README.md       # Documentation
└── .gitignore      # Fichiers ignorés
```

## 💡 Comment ajouter des questions

Ouvre `index.html` et modifie la fonction `genererQuestions()` pour ajouter des questions réelles.

## 🎮 Utilisation

1. Choisis ton niveau (6ème, 4ème, 3ème)
2. Sélectionne une matière
3. Réponds à 20 questions
4. Vois ton score

Enjoy! 🎉
